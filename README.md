# QC Weight System - ระบบ QC น้ำหนักสินค้า

ระบบบันทึกและตรวจสอบน้ำหนักสินค้าประจำเครื่องจักรในโรงงาน สำหรับ QC ตรวจน้ำหนักทุกชั่วโมง พร้อมสรุปผลงานพนักงานคุมเครื่องและความถี่การตรวจของ QC

## เข้าใช้งาน

https://mhuyong.github.io/QC-Weight-System/

## ฟีเจอร์หลัก

### บันทึกน้ำหนัก
- QC กรอกข้อมูล: เครื่องจักร, สินค้า, พนักงานคุมเครื่อง, น้ำหนัก
- แสดงพิกัดน้ำหนัก (ต่ำสุด-สูงสุด) ของสินค้าที่เลือก
- ตรวจสอบสถานะทันที (ผ่าน / ต่ำกว่าเกณฑ์ / เกินเกณฑ์)
- ดึงข้อมูลชั่วโมงก่อนหน้าอัตโนมัติ (ชื่อคนคุมเครื่อง, QC, สินค้า)
- เปลี่ยนชื่อพนักงานคุมเครื่องได้กรณีเปลี่ยนกะ

### ข้อมูลบันทึก
- ดูข้อมูลที่บันทึกทั้งหมด
- กรองตามวันที่และชั่วโมง
- แสดงสถานะผ่าน/ไม่ผ่านด้วย badge สี

### สรุปผล
- **ผลงานพนักงานคุมเครื่อง**: อัตราผ่าน (%), จำนวนครั้งต่ำกว่า/เกินเกณฑ์, ระดับ (ดีเยี่ยม/ปานกลาง/ต้องปรับปรุง)
- **รายละเอียดรายชั่วโมง**: แสดงเครื่องไหน พนักงานคนไหน ที่ขาดหรือเกินในชั่วโมงนั้นๆ
- **ความถี่การตรวจ QC**: ตรวจครบทุกเครื่องทุกชั่วโมงหรือไม่ แสดง % ความครอบคลุม

### ตั้งค่าข้อมูลหลัก
- จัดการเครื่องจักร (เลขเครื่อง, ชื่อเครื่อง)
- จัดการสินค้า (ชื่อ, น้ำหนักเป้า, ต่ำสุด, สูงสุด, หน่วย)
- จัดการพนักงานคุมเครื่อง (ชื่อ, รหัสพนักงาน)
- จัดการ QC ผู้ตรวจ (ชื่อ, รหัสพนักงาน)
- เพิ่ม / แก้ไข / เปิด-ปิดการใช้งาน

