# Product Weight Monitoring & Performance Tracking System

ระบบบันทึกน้ำหนักสินค้าและติดตามประสิทธิภาพการทำงานของพนักงานควบคุมเครื่องจักร (Operator) และเจ้าหน้าที่ควบคุมคุณภาพ (QC) เพื่อใช้ในการประเมินผลตอบแทน (Incentive / KPI)

## 📌 ภาพรวมระบบ (Project Overview)
ระบบนี้พัฒนาขึ้นเพื่อแก้ไขปัญหาและควบคุมคุณภาพน้ำหนักสินค้าของเครื่องจักรแต่ละเครื่องให้อยู่ในพิกัดที่กำหนด (Standard Weight) โดยมีเจ้าหน้าที่ QC คอยสุ่มตรวจทุกชั่วโมง ระบบจะทำการเชื่อมโยงข้อมูลผู้ควบคุมเครื่องจักรและ QC จากชั่วโมงก่อนหน้าเพื่อความต่อเนื่องในการทำงาน และสรุปผลลัพธ์ออกมาเป็นรายงานสำหรับการคิดผลตอบแทนอย่างแม่นยำ

---

## ✨ คุณสมบัติเด่น (Key Features)

### 1. ระบบบันทึกข้อมูลการตรวจน้ำหนักรายชั่วโมง (Hourly QC Inspection)
* **บันทึกข้อมูลพื้นฐาน:** ชื่อผู้ควบคุมเครื่อง, หมายเลขเครื่องจักร, น้ำหนักสินค้าที่วัดได้
* **ระบบดึงข้อมูลอัตโนมัติ (Auto-populate):** ในชั่วโมงถัดไป ระบบจะดึงข้อมูล **ชื่อผู้ควบคุมเครื่องจักร** และ **ชื่อ QC** จากชั่วโมงก่อนหน้ามาแสดงให้อัตโนมัติ เพื่อลดเวลาในการคีย์ข้อมูล
* **รองรับการเปลี่ยนกะ (Shift Change Support):** สามารถแก้ไข/เปลี่ยนแปลงชื่อผู้ควบคุมเครื่องจักรได้ทันทีหากมีการเปลี่ยนคนคุมเครื่องระหว่างชั่วโมง

### 2. หน้าแสดงผลข้อมูลดิบ (Data Logs View)
* หน้ารวมรายการข้อมูลทั้งหมดที่มีการบันทึกเข้าระบบ สามารถตรวจสอบย้อนหลังได้

### 3. หน้าสรุปผลและรายงานประสิทธิภาพ (Dashboard & Summary Report)
* **รายงานเครื่องจักรและพนักงาน (Operator Performance):** สรุปจำนวนครั้งที่น้ำหนักสินค้า **"ขาด"** หรือ **"เกิน"** เกณฑ์มาตรฐาน แยกตามรายเครื่องจักรและรายบุคคลในแต่ละชั่วโมง
* **รายงานการปฏิบัติงานของ QC (QC Audit Frequency):** ตรวจสอบความถี่ในการเข้าตรวจของ QC ว่าได้เข้าตรวจครบทุกเครื่อง "ทุกชั่วโมง" จริงหรือไม่
* **ระบบสนับสนุนการจ่ายผลตอบแทน (Incentive Calculation Support):** สรุปข้อมูลเชิงสถิติเพื่อนำไปใช้คำนวณโบนัสหรือหักคะแนนตามผลงานจริง (KPI)

## 🛠 โครงสร้างข้อมูลที่สำคัญ (Data Structure)

### ข้อมูลพิกัดสินค้า (Product Specification)
* `Product_ID` : รหัสสินค้า
* `Min_Weight` : น้ำหนักขั้นต่ำที่ยอมรับได้
* `Max_Weight` : น้ำหนักสูงสุดที่ยอมรับได้

### ข้อมูลบันทึกการตรวจ (Weight Log Table)
* `Timestamp` : วันที่และเวลาที่ตรวจ (ชั่วโมงที่ตรวจ)
* `Machine_No` : เลขเครื่องจักร
* `Operator_Name` : ชื่อคนคุมเครื่อง
* `QC_Name` : ชื่อ QC ที่ตรวจ
* `Recorded_Weight` : น้ำหนักสินค้าที่วัดได้
* `Status` : สถานะน้ำหนัก (`ปกติ` / `ขาด` / `เกิน`)
